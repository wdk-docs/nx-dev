# report

Reports useful version numbers to copy into the Nx issue template

## Usage

```bash
nx report
```

[Install `nx` globally](/getting-started/nx-setup#install-nx) to invoke the command directly using `nx`, or use `npx nx`, `yarn nx`, or `pnpx nx`.

## Options

### help

Show help

### version

Show version number
